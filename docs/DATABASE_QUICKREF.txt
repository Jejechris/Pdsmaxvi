HAPUS 1 ENTRY:
--------------
1. Hover pada entry
2. Klik icon ❌ di kanan
3. Confirm

HAPUS SEMUA:
-----------
1. Klik path /leaderboard
2. Klik icon ⋮ (three dots)
3. Pilih "Delete"
4. Confirm
5. SEMUA DATA TERHAPUS!

🔒 MATIKAN DATABASE SETELAH EVENT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

OPSI 1 - Lock (Data tetap ada tapi tidak bisa diakses):
-------------------------------------------------------
1. Tab "Rules"
2. Ganti jadi:
   {
     "rules": {
       ".read": false,
       ".write": false
     }
   }
3. Publish

OPSI 2 - Delete Database (Permanent):
-------------------------------------
1. Settings (⚙️)
2. Delete database
3. Confirm
